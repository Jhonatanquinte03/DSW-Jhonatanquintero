CSS Grid

CSS Grid es un sistema de maquetación web que divide la página en una cuadrícula o rejilla a partir de la cual se pueden posicionar los diferentes elementos de manera más sencilla, versátil y coherente. Ademas Es fácil de usar y sensible por diseño. Y de Por supuesto, uno puede mostrar los elementos de la página en cualquier orden, sin depender de ninguna manera del orden descrito en código fuente.

Niess, G., Roubal, A., Thurner, S., & Roque, E. B. (2019). CSS Grid Layouts.
